# Snake

Classic Snake game written in C with SDL2.

## Requirements

- GCC
- SDL2

```bash
# Ubuntu / Debian
sudo apt install gcc libsdl2-dev
```

## Build

```bash
make
```

## Run

```bash
./snake
```

## Controls

| Action | Keys |
|--------|------|
| Move   | Arrow keys / ZQSD / WASD |
| Start / Restart | Enter or Space |
| Quit   | Escape |

## Features

- Increasing speed as score grows
- Best score tracked per session
- Score displayed in the window title
